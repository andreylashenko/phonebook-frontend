# Телефонный справочник

Тестовое задание компании ProfitClick. Frontend часть.

Развертка проекта: в корне проекта выполнить следующие команды

``` bash
# install vue/cli
> npm install -g @vue/cli
# install dependencies
> npm install
# build project
> npm run build
# start project
> npm run dev
```
